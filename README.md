## 查询
- 查询某个员工的直属上级
- 查询某个经理管理的所有下属
- 查询某段时间内新入职的员工
- 查询某个项目的项目经理姓名、客户信息、以及参与部门名称
- 统计每个员工的考勤记录数量或缺勤次数
- 查询某员工的薪资发放历史
- 查询某个客户所有项目对应的项目经理
- 查询某个部门参与的所有项目及其客户名称
- 查询所有姓为某某的员工
- 统计参与某任务的所有员工及其角色
- 查询所有涉及（worker or manager）某位员工的项目
- 查询没有被分配任何任务的员工
- 查询没有下属的员工
